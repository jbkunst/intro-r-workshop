Introducción al Uso de R para Análisis Estadístico

¿Por qué R?

  • Lenguaje orientado a los datos (*)
  • Flexibilidad. Lenguaje de interfaces.
  • Comunidad

Antes de partir

  • Presentación http://bit.ly/intro-r-puc
  • Código http://bit.ly/intro-r-puc-codigo
  • Asignación: var <- 5 significa que la variable var contiente el valor 5
  • Pipe: var %>% f() que a la variable var le aplico la función f. Leer como luego
  • De la misma forma: var %>% f() %>% g() a var le aplico f, luego g

Antes de partir

  • Tabla, Datos tabulados o _data frame es un conjunto de valores cada uno asociado a una variable y una observación

En general cualquier análsis(*)

  • APIs
